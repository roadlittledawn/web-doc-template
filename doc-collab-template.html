Structure > Taxonomy

Best for renaming or editing terms [editing how? add to section title?].

To rename a term:

  1. From Structure > Taxonomy > Primary Navigation > list terms, select edit for the desired term.
  2. Rename the term using the Name field.
  3. Update the URL Path Settings > URL alias field to match the new name (no articles, dashes instead of spaces).
  4. Review affected docs' URLs and create redirects (start at step 3) as necessary for all docs associated with that URL path.
Structure > Taxonomy Manager

Best for moving or adding terms. [Not always stable?]

To move a term:

  1. From Structure > Taxonomy Manager > Primary Navigation, select the checkbox of the term you want to move.
  2. Select Move from the Toolbar and start typing name of new parent term. Select the term from the autosuggest list.
  3. Select Move [parent option important?].
  4. Review affected docs' URLs and create redirects (start at step 3) as necessary for all docs associated with that URL path.

To add a term:

  1. From Structure > Taxonomy Manager > Primary Navigation, select a term checkbox to define the parent of your new term.
  2. Enter the name of the new term (as it will appear on front-end).
  3. Select Add. The term is added to the regenerated list. It will appear in associated menus when a doc is added to that term.
